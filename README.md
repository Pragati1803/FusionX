# FusionX

A real-time collaborative code editor built with React, Tailwind CSS, Node.js, Express & Socket.IO.

✨ Users can create rooms, edit code together, chat, manage files, and run code in real time.

## Live Demo
*(Deploy this yourself after pushing to GitHub & Vercel/Render/Heroku)*

## 💻 Tech Stack
- React + Vite
- Tailwind CSS
- TypeScript
- Node.js & Express
- Socket.IO
- Piston (code execution backend)
